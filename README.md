# jijinhee_twitter
지진희알림 (feat. twitter)



1. 텔레그램 봇 만들기

	- 구글 검색 : 텔레그램 봇 만들기

2. 트위터 API 얻기

	- 구글 검색 : 트위터 api key 발급
	
3. node.js 환경 구축

	- 구글 검색 : node.js 서버 구축

4. 필요한 모듈 설치

# npm install node-telegram-bot-api
# npm install twitter

5. node.js 프로그램 하기 - jijinhee_twitter.js

6. 실행

# node jijinhee_twitter.js

or 백그라운드에서 실행하려면
# npm install forever
# forever start jijinhee_twitter.js
